# 🕒 U-Time

Веб-сайт по тайм-менеджменту, позволяющий пользователям расставлять приоритеты и максимально эффективно выполнять задания в отведённые сроки.

## О проекте

**U-Time** — это интеллектуальный инструмент для управления задачами, который помогает пользователям организовывать своё время и повышать продуктивность.

### Ключевые возможности

- 📝 **Создание задач** с планированием на любую дату
- 🎯 **Расширенная матрица Эйзенхауэра** для определения приоритетов задач (важно/срочно)
- 📊 **Динамичные шкалы времени** — визуальное отображение оставшегося времени до дедлайна
- 📈 **Персонализированная статистика** выполнения задач

Проект построен на архитектурном шаблоне **MVC** (Model-View-Controller) с разделением на два независимых компонента:

#### 📦 **Модель (Model)**
- Отвечает за данные и бизнес-логику приложения
- Устанавливает соединение с базой данных
- Предоставляет данные в формате JSON для внешних запросов
- Содержит валидацию и правила обработки информации

#### 🎨 **Представление (View)**
- Отвечает за отображение данных пользователю
- Формирует HTML страницы с динамическим контентом
- Обеспечивает адаптивный и интуитивно понятный интерфейс

#### 🎮 **Контроллер (Controller)**
- Обрабатывает входящие HTTP запросы от пользователя
- Взаимодействует с API для получения или отправки данных
- Выполняет валидацию пользовательского ввода


## Архитектура продукта
Для лучшей организации кода и разделения ответственности контроллеры, отвечающие за взаимодейтсвие с пользователем (MVC), и контроллеры, отвечающие за взаимодействие с базой данных (API), были разделены по разным директориям. Таким образом, данный продукт имеет следующую архитектуру:

#### 🎯 **UtimeWEB**
- Обработка пользовательского ввода
- Взаимодействие с пользовательским интерфейсом
- Отправка запросов к API
- Отображение полученных данных

#### ⚙️ **UtimeAPI**
- Установление соединения с базой данных
- Взаимодействие с базой данных
- Предоставление данных в формате JSON
- Бизнес-логика приложения
