# UtimeAPI
## 🕒 О проекте
U-Time - веб-сайт по тайм-менеджменту, позволяющий пользователям расставлять приоритеты и максимально эффективно выполнять задания в отведённые сроки. Благодаря данному веб-сайту пользователь может создавать задачи, запланированные на любую дату, определять приоритет задачи на основе матрицы Эйзенхауэра - простого, но в то же время эффективного метода тайм-менеджмента. Отличительной особенностью веб-сайта U-Time является отображение динамичных шкал, наглядно показывающих пользователю оставшееся время до наступления дедлайна задачи. Также пользователь может просматривать персонализированную статистику о выполнении задач.

Для разработки веб-сайта использовался шаблон MVC (Модель-Представление-Контроллер):
* **Модель**: отвечает за данные, методы работы с ними и структуру программы;
* **Представление**: отображает данные на уровне пользовательского интерфейса;
* **Контроллер**: обеспечивает взаимодействие с системой.
## Архитектура продукта
Для лучшей организации кода и разделения ответственности контроллеры, отвечающие за взаимодейтсвие с пользователем (MVC), и контроллеры, отвечающие за взаимодействие с базой данных (API), были разделены по разным директориям. Таким образом, данный продукт имеет следующую архитектуру:
* **UtimeAPI**: Установление соединения и взаимодействие с базой данных, возвращение данных, запрашиваемых со стороны пользователя, в формате JSON;
* **UtimeWEB**: Обработка пользовательского ввода, запросы к UtimeAPI, получение и обработка возвращённых данных.
